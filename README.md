# 5-Bit-CLA-Carry-Look-Ahead-Adder
This project implements a 5-bit Carry Look-Ahead (CLA) Adder designed using VLSI concepts. The circuit is built from basic logic gates such as inverters, NAND gates, and carry generation blocks. Layout and simulations are performed using Magic tools, demonstrating faster addition by reducing carry propagation delay compared to ripple carry adders.
